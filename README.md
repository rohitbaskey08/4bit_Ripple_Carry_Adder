Ripple Carry Adder (RCA) – Verilog
📌 Overview

This project implements a 4-bit Ripple Carry Adder (RCA) using Verilog HDL.
A Ripple Carry Adder performs binary addition by connecting multiple Full Adders in series. The carry output of one full adder becomes the carry input of the next full adder.

Because the carry propagates through each stage sequentially, it is called a Ripple Carry Adder.
